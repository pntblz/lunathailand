## 🌟 ภาพรวมโปรเจกต์ (Project Overview)

เว็บไซต์นี้ถูกออกแบบและพัฒนาขึ้นใหม่ทั้งหมดเพื่อแบรนด์ **MO LUNA OFFICIAL** โดยเปลี่ยนโฉมจากดีไซน์เดิมให้มีความเป็น **Luxury Beauty Brand** อย่างเต็มตัว เน้นการใช้โทนสีอบอุ่น (Warm Cream & Rose Gold) และความเรียบหรูสะอาดตา พร้อมอนิเมชันที่ลื่นไหลเพื่อมอบประสบการณ์พรีเมียมให้แก่ผู้เข้าชม

### ✨ ฟีเจอร์หลัก (Key Features)
- 💅 **Luxury UI/UX Design:** ดีไซน์ที่เน้นความเรียบหรู พื้นหลังสไตล์ Glassmorphism ผสมผสานกับ Ambient Gradients สุดละมุน
- 🚀 **Smooth Scroll Animations:** เพิ่มชีวิตชีวาด้วย `Framer Motion` ทุกส่วนของเว็บจะค่อยๆ Fade-in และ Slide-up อย่างนุ่มนวลเวลาเลื่อนจอ
- 📱 **Mobile-First & Floating CTA:** รองรับการแสดงผลทุกหน้าจออย่างสมบูรณ์แบบ พร้อมปุ่ม "ช้อปผ่าน Shopee" ลอยติดหน้าจอตลอดเวลาบนมือถือ (Floating Action Button) เพื่อกระตุ้นยอดขาย
- 🏷️ **Dynamic Product Badges:** ป้ายกำกับสินค้าสุดโดดเด่น (เช่น `BEST SELLER`, `NEW`, `HOT`, `PREMIUM`) ดึงดูดสายตาลูกค้าได้ตรงจุด
- ⚡ **Ultra-Fast Performance:** ใช้ Next.js ร่วมกับ `output: "export"` ทำให้เว็บเบา โหลดไวปรู๊ดปร๊าดแบบ Static Site 100%
- 🔗 **Social Link Preview:** ตั้งค่า Open Graph (OG Image) สมบูรณ์แบบ เมื่อแชร์ลิงก์ลง LINE หรือ Facebook รูปโลโก้แบรนด์จะขึ้นโชว์อย่างสวยงาม

## 🚀 การรันโปรเจกต์ (Local Development)

หากต้องการนำโค้ดไปพัฒนาต่อหรือรันบนเครื่องตัวเอง สามารถทำตามขั้นตอนต่อไปนี้:

### 1. ติดตั้ง Dependencies
```bash
npm install
```

### 2. รันเซิร์ฟเวอร์
```bash
npm run dev
```
เปิดเบราว์เซอร์ไปที่ [http://localhost:3000](http://localhost:3000)

### 3. คำสั่งสำหรับการ Build นำไปใช้งานจริง (Production Build)
โปรเจกต์นี้ตั้งค่า `next.config.ts` ให้ทำ **Static Export** เรียบร้อยแล้ว
```bash
npm run build
```
เมื่อรันคำสั่งนี้เสร็จ จะได้โฟลเดอร์ `/out` ที่สามารถนำไปอัปโหลดขึ้นโฮสติ้งไหนก็ได้ที่เป็น Static Hosting (เช่น Cloudflare Pages, GitHub Pages)

## ☁️ การนำขึ้น Cloudflare Pages (Deployment Guide)

โปรเจกต์นี้ถูกออกแบบมาให้ทำงานร่วมกับ Cloudflare Pages ได้อย่างสมบูรณ์แบบโดยไม่ต้องตั้งค่าให้ปวดหัว:

1. นำโค้ดทั้งหมดขึ้น GitHub Repository
2. เข้าไปที่ Cloudflare > **Workers & Pages**
3. สร้างโปรเจกต์ใหม่และเชื่อมกับ GitHub Repo ของคุณ
4. กำหนดค่า Build ดังนี้:
   - **Framework preset:** `Next.js (Static HTML Export)`
   - **Build command:** `npm run build`
   - **Build output directory:** `out`
5. *ระบบจะดึงเวอร์ชัน Node 20 จากไฟล์ `.nvmrc` ไปใช้งานให้โดยอัตโนมัติ ไม่ต้องตั้ง Environment Variables เพิ่ม*
